Free Deployment

  1. Push this `worldcup-live` folder to GitHub.
  2. Create a Cloudflare Pages project from the repo.
  3. Use `public` as the output directory.
  4. Add API secrets only if you want automatic live scores.

Live API

The app calls `/api/live`. The included Pages Function hides your API key and caches responses, so visitors do not burn through the free API quota individually.

FOOTBALL_PROVIDER=api-football
API_FOOTBALL_KEY=your_key
API_FOOTBALL_LEAGUE=1
API_FOOTBALL_SEASON=2026

Refresh Data From Excel

python3 scripts/export_workbook.py "/path/to/2026 FIFA World Cup.xlsx" --output public/data/predictions.json
